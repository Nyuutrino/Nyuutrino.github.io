Stargaze

An interactive Stargazing simulator. Downloadable as a PWA (web app), and fully functional offline (as well as having cache control), this was a month long project that took a lot of effort and I am very proud of.

CLIp

CLIp was my attempt at making a CLI language. It was pretty fun and for a while I was working hard on it. Alas, this project didn't last long. However, its remanents serves as a valuable reminder of how much I learned. The website was my first large project in which I used a server language (Go) extensively, and the language itself helped me understand more about how a programming language works behind the scenes and allowed me to better appreciate its intricate and beautiful nature. I enjoyed the journey very much and was glad that I made it.

Fake BSoD
